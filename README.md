# Main Page for [RateLink](https://www.rate-link.com)

## Features

- Input shipping freight rate with few clicks
- Input multiple data with single save action
- Search rates by
  - Input Person
  - Shipper
  - Liner
  - Place of Loading
  - Place of Discharging
